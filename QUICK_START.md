## First Time Setup (Do This First!)

### Step 1: Create Project Structure

Run this command in your project folder:

```bash
python setup_project.py
```

This creates all necessary folders:
- `raw_data/` - For raw CSV files
- `aggregated_data/` - For processed files
- `logs/` - For processing logs
- `data/` - For Streamlit dashboard
- `pages/` - For additional dashboard pages

---

### Step 2: Generate Sample Data

```bash
python generate_sample_data.py
```

This creates test data in `raw_data/` folder.

---

### Step 3: Process Data

```bash
python ecommerce_data_processor.py
```

This creates aggregated files in `aggregated_data/` folder.

---

### Step 4: Copy Data for Dashboard

**Windows:**
```bash
copy aggregated_data\*.csv data\
```

**Mac/Linux:**
```bash
cp aggregated_data/*.csv data/
```

---

### Step 5: Run Dashboard

```bash
streamlit run app.py
```

Dashboard opens at: http://localhost:8501

## âš ï¸ Common Errors & Fixes

### Error: "FileNotFoundError: logs/aggregation_log.txt"

**Cause:** Missing folders

**Fix:** Run setup script first
```bash
python setup_project.py
```

---

### Error: "FileNotFoundError: raw_data/user_table.csv"

**Cause:** No data files

**Fix:** Generate sample data
```bash
python generate_sample_data.py
```

---

### Error: Streamlit can't find data files

**Cause:** Data is in wrong folder

**Fix:** Copy aggregated data to `data/` folder
```bash
# Windows
copy aggregated_data\*.csv data\

# Mac/Linux
cp aggregated_data/*.csv data/
```

---

### Error: "ModuleNotFoundError: No module named 'streamlit'"

**Fix:** Install dependencies
```bash
pip install -r requirements.txt
```

## ğŸ› Still Having Issues?

1. **Check Python version:** Should be Python 3.8+
   ```bash
   python --version
   ```

2. **Reinstall dependencies:**
   ```bash
   pip install --upgrade -r requirements.txt
   ```

3. **Check file permissions:** Make sure you can write to the folder

4. **Check the logs:** Look in `logs/aggregation_log.txt` for errors

5. **Run each script individually** to see where it fails:
   ```bash
   python setup_project.py
   python generate_sample_data.py
   python ecommerce_data_processor.py
   ```

## ğŸ’¡ Pro Tips

1. **Don't delete folders** - The setup script won't overwrite existing data

2. **Check logs regularly** - `logs/aggregation_log.txt` tells you what happened

3. **Data updates** - Just re-run the processor, it will overwrite old aggregated files

4. **Dashboard not updating?** - Streamlit caches data. Refresh the page or restart Streamlit
